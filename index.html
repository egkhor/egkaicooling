🌡 WET-BULB PHYSICS — WHY IT LIMITS EVAPORATIVE COOLING

Evaporative cooling cannot cool air below its wet-bulb temperature (WBT). This is a hard thermodynamic ceiling, not an engineering choice. In Malaysia, WBT ranges 24–27°C depending on season and time of day — leaving only a narrow driving force.

// WET-BULB TEMPERATURE (psychrometric approx.)
T_wb ≈ T_db × atan[0.151977·(RH+8.313659)^0.5]
    + atan(T_db + RH) − atan(RH − 1.676331)
    + 0.00391838·RH^1.5 · atan(0.023101·RH) − 4.686035
// Stull (2011) — accurate ±0.65°C for 5%≤RH≤99%
// MALAYSIA CASE: worst-case monsoon peak
T_db = 32°C · RH = 90%
T_wb ≈ 27.1°C
// MALAYSIA CASE: typical daytime
T_db = 30°C · RH = 75%
T_wb ≈ 25.1°C
⟹ Evaporative cooling ceiling: 25–27°C inlet air — NOT 18–20°C as in arid climates

📉 EVAPORATIVE EFFICIENCY DERATING — MALAYSIA RH

Direct evaporative cooler efficiency η = (T_db_in − T_db_out) / (T_db_in − T_wb_in). At high RH, the driving potential collapses. The wet wall's −6 to −8°C claim in v1 was for dry conditions only.

📊 PSYCHROMETRIC OPERATING WINDOW — MALAYSIA

Key finding: Malaysia's typical operating range (RH 75–88%, T 28–33°C) places the system predominantly in Zone 2 — Hybrid Mode. Pure passive operation is achievable during cooler mornings, night hours, and the drier inter-monsoon period (March–April, Sep–Oct). The original "Zero AC" claim is physically untenable for a high-density AI server at Malaysia's average humidity.

Deep Q-Network agent augmented with a real-time psychrometric guard layer — new in v3. Before issuing any evaporative cooling action, the agent checks T_wb against the current derating table. If η_evap < 30%, wet wall is bypassed and the backup unit quota is evaluated.

Q(s,a) ← Q(s,a) + α·[r + γ·max Q(s',a') − Q(s,a)]
// α=0.001 · γ=0.97 · ε-greedy exploration
// REWARD FUNCTION (v3 — includes backup penalty)
R = −w₁·(T_chip−T_set)²

   −w₄·P_backup_unit
